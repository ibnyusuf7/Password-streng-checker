# Security Log Analyzer

A Python script that analyzes system log files to detect suspicious activity.

## Description

Log files contain valuable security information.  
This tool scans logs to detect possible threats such as:

- Failed login attempts
- Suspicious IP addresses
- Repeated access attempts

## Features

- Reads log files
- Detects brute force login attempts
- Counts failed login entries
- Displays suspicious activity

## Technologies

- Python
- File handling
- Basic threat detection logic

## Usage

python analyzer.py log.txt
